# 导航

## 人生·江湖（黄沾）

天下风云出我辈，  
一入江湖岁月催；  
皇图霸业谈笑间，  
不胜人生一场醉。  
提剑跨骑挥鬼雨，  
白骨如山鸟惊飞；  
尘事如潮人如水，  
只叹江湖几人回。
