## 注意

1. `goroutine`

   ```go
   func main() {
   	names := []string{"Eric", "Jim", "John"}
   	for _, v := range names {
   		/*
   		正确的写法
   		go func(v string) {
   		  	fmt.Println(v)
   		}(v)
   		*/
   		go func() {
   			fmt.Println(v)
   		}()
   	}
   	time.Sleep(time.Millisecond)
   }
   
   /**
   	结果：
   	  John
       John
       John
   */
   ```
